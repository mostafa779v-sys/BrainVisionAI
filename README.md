# BrainVisionAI

BrainVisionAI is a deep learning project for automatic brain MRI tumor classification.

## Features

- Brain MRI Classification
- EfficientNet-B0
- PyTorch
- Google Colab Support
- 95%+ Test Accuracy
- GitHub Version Control

## Dataset

Brain Tumor MRI Dataset (4 Classes)

- Glioma
- Meningioma
- No Tumor
- Pituitary
